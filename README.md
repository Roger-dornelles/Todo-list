Criação do Todo-list

LINK:   https://todo-list-zeta-eight.vercel.app/

Tecnologias usadas na criação:


React.js e Typescript

Todo-list criado para adicionar tarefas a serem feitas no decorrer do dia, assim que concluida pode-se fazer a marcação de concluida assim ela sera riscada para mostrar sua conclusão.


Futuramente sera adicionado o back-end para o armazenamento das informações para consultas quando necessario.
